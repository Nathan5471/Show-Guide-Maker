# Show Playlist Maker

This show playlist maker copies files an makes it switch through different shows while still going in order

## Installation and running
Simply clone the repo and run showPlaylistMaker.py

## Storing your files
> Show folder
> > Show name
> > > Season 01
> > > > Episode S01E01.ext \
> > > > Episode S01E02.ext \
> > > > Episode S01E03.ext
